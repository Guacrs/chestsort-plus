# Chestsort+

Sorting is enabled for everyone by default — no permissions or setup required.

## How to sort

**Chest / barrel / shulker (all players):**
- Open the container. If slot 8 (top-right of the first row) is empty, a green **Sort** button appears there. Click it to sort.
- If slot 8 already has items, tap **hotbar slot 8** (rightmost hotbar slot) instead.

**Bedrock / Geyser players:** The hotbar slot 8 tap is the most reliable option. Chest GUIs can be awkward on mobile, but the labeled Sort button in slot 8 usually works when that slot is free.

**Java players:** Can also shift-click an empty slot to sort (configurable via `/chestsort`).

## Settings

Run `/chestsort` to toggle sorting on/off or change the Java shift-click keybind.

Admin commands (`chestsort reload`, `chestsort version`) require `chestsort.admin` (operators by default).
